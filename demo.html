🌐 SoCump

Social App Browser Detection and Redirect

检测社交应用内置浏览器并引导用户跳转到外部浏览器

📱 功能特性

  • 自动检测微信、QQ、支付宝、微博等社交应用内置浏览器
  • 智能引导用户跳转到外部浏览器
  • 支持URL参数传递目标链接
  • 一键复制链接到剪贴板
  • 响应式设计，适配移动端
  • 无依赖，纯JavaScript实现
  • 支持自定义样式和文案

🚀 快速开始

将以下代码添加到您的HTML页面中：

<script src="socump.js"></script>

或者通过URL参数指定目标链接：

https://yoursite.com/index.html?url=https://target-site.com

⚙️ 高级用法

您可以通过JavaScript API进行更多自定义：

// 创建实例 const socump = new SoCump(); // 检测浏览器类型 if (socump.isWeChat()) { console.log('在微信中打开'); } // 手动触发跳转 socump.openInBrowser(); // 复制链接 socump.copyToClipboard('https://example.com');

📝 使用说明

  1. 自动检测：脚本会自动检测当前是否在社交应用内置浏览器中
  2. 智能跳转：如果不在社交应用中，会自动跳转到目标URL
  3. 用户引导：如果在社交应用中，会显示引导页面
  4. 一键复制：用户可以一键复制链接到剪贴板
  5. 手动打开：尝试在外部浏览器中打开链接
